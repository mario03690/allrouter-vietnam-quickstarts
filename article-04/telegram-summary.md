## Long
📝 Cách debug lỗi 403/429/timeout khi dùng AI coding tools từ Việt Nam

Khi AI coding workflow bị gãy, phản xạ thường gặp là đổi luôn VPN, đổi provider, đổi model, đổi CLI. Vấn đề là khi thay mọi thứ cùng lúc, bạn gần như mất khả năng biết lỗi nằm ở đâu.

Bài viết đi qua:
- 403 thường gợi ý vấn đề auth / permission / access path
- 429 thường liên quan quota / rate / usage pattern
- timeout thường nghiêng về network, request shape hoặc workflow length
- thứ tự test tối thiểu để debug nhanh hơn và ít sửa nhầm hơn
